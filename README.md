# Diamond Price Prediction

## Overview
This project predicts the price of a diamond using Machine Learning.

The model was developed using the CatBoost Regressor and deployed using Streamlit.

## Features
- Predict diamond prices
- User-friendly Streamlit interface
- CatBoost Regressor model
- Real-time prediction

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- CatBoost
- Streamlit
- Joblib

## Input Features
- Carat
- Cut
- Color
- Clarity
- Depth
- Table
- x
- y
- z

## Model
CatBoost Regressor

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
